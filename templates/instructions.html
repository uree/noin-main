Noin

A coin that says nein.

---------------------------------

Set it up


Android:

  1. Download PhonePI app
  2. Enable PhonePI to run in the background (in: battery settings)
  3. In PhonePI connect to noin server (IP: 192:168:0:116 Port: 5000)
  4. Turn accelerometer on in PhonePI
  5. Start "mining noin" (noining)
